Installation

For installing the script from Moho itself, go to "Scripts -> Install Script..." and just follow the instructions. :

  1. Press "Select Script Folder" button.
  2. Browse to where you uzipped the downloaded file and select ls_shapes_window folder.
  3. Well, that should be all.
